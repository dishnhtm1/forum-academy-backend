# Forum Academy Portal – Backend

This is the **Node.js + Express backend** for the Next-Gen Academic Portal. It exposes API endpoints for course listings, applications, login, and news.

## 🔧 Tech Stack

- 🟢 Node.js + Express
- 🛢️ MongoDB (via Mongoose)
- 🔐 JWT Authentication
- 🔄 CORS-enabled
- 📁 Environment config with dotenv
- 🚀 Hosted on Render or Railway
